DungeonKeep Escape

Embark on a keyboard-only platformer in a pixelated fortress. No enemies, just precise navigation through puzzles and platforms.

Controls

  • W key - Jump, Interact
  • A key - Sprint left
  • D key - Sprint right

Task

Go through every door and beat each level.

About the game

JavaScript handles the coding, while the Tiled application seamlessly weaves together detailed maps. This uncomplicated yet potent pairing ensures a seamlessly running game with an aesthetically pleasing environment.
